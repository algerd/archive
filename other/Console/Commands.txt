
Копирование/вставка: 
	- сохранить в буфере из консоли: ЛКМ выделить, ПКМ - сохранить в буфер 
	- копирование из буфера в консоль: ПКМ по месту куда надо скопировать (также копировать и из редактора)
	- копировать в редактор: ctr+v или ПКМ-вставить
	

Горячие клавиши в консоли:
	Ctrl+C - прерывание команды
		
Пробелы являются разделителями команд.
	
Команда вызова помощи по всем командам: help	
	Команда вызова помощи по какой-нибудь команде: help [<команда>]	
	C:\Users\Александр>help cd

Очищает содержимое экрана: CLS	
	d:\>help cls
			
Чтобы учитывались пробелы в именах файлов и каталогов при отключении расширенной обработки команд использовать полные команды:
	chdir, mkdir, rmdir
В случае использования сокращённых команд при отключении расширенной обработки команд использовать обрамление пути директории ковычками:	
	c:\>cd "\Program Files" 
		
Вывод списка файлов и подкаталогов в указанном каталоге:
	DIR [диск:][путь][имя файла] [/A[[:]атрибуты]] [/B] [/C] ... - метки и атрибуты - это фильтры вывода (сортировки, вид вывода и т.д. полный список см в help)
		/A          Отображение файлов с указанными атрибутами.
					 D  Каталоги.            R  Файлы, доступные только для чтения.
					 H  Скрытые файлы.       A  Файлы, готовые для архивирования.
					 S  Системные файлы.     I  Файлы с неиндексированным содержимым.
		...
			c:\>dir			 
			c:\>dir /a:h	
			c:\>dir d:\софты /b			- с указанием пути директории
			c:\>dir d:\софты /o:d     	- сортировки /O:атрибут сортировки
	
Команда навигации по файловой системе:
	CHDIR [/D] [диск:][путь] - для одновременной смены текущих диска и каталога
	CHDIR [..] 				 - для переходов по текущему диску 
	CD [/D] [диск:][путь]	 - для одновременной смены текущих диска и каталога 
	CD [..] 				 - для переходов по текущему диску 
	 ..  обозначает переход в родительский каталог.
		c:\>cd \temp
		c:\temp>cd /d d:\Софты
		d:\Софты>cd ..
		d:\> 
		c:\>cd "\Program Files"  - воспринимается всё имя внутри "" вместе с пробелами
		c:\>chdir \Program Files  
	
Создание каталога:
	MKDIR [диск:][путь] - Команда MKDIR создает при необходимости все промежуточные каталоги в пути.
	MD [диск:][путь] - При отключении расширенной обработки команд используется только второй вариант.	  		
		d:\>mkdir \test\test1\test2
		d:\>md folder1\folder2
		d:\>cd \folder1\folder2
		d:\folder1\folder2>
 
Удаление каталога:
	RMDIR [/S] [/Q] [диск:]путь
	RD [/S] [/Q] [диск:]путь
		/S      Удаление дерева каталогов, т. е. не только указанного каталога, но и всех содержащихся в нем файлов и подкаталогов.
		/Q      Отключение запроса подтверждения при удалении дерева каталогов с помощью ключа /S. 
			d:\>rd \folder1		 - просто rd удаляет только пустые папки
				Папка не пуста.
			d:\>rd /s \folder1
				\folder1, Продолжить [Y(да)/N(нет)]? y
			d:\>rd /s /q \test	- удаляет каталог без подтверждения

Копирование одного или нескольких файлов в другое место.
	COPY [/D] [/V] [/N] [/Y | /-Y] [/Z] [/L] [/A | /B] источник [/A | /B]
     [+ источник [/A | /B] [+ ...]] [результат [/A | /B]]

	  источник     Имена одного или нескольких копируемых файлов.
	  /A           Файл является текстовым файлом ASCII.
	  /B           Файл является двоичным файлом.
	  /D           Указывает на возможность создания зашифрованного файла результат  Каталог и/или имя для конечных файлов.
	  /V           Проверка правильности копирования файлов.
	  /N           Использование, если возможно, коротких имен при копировании файлов, чьи имена не удовлетворяют стандарту 8.3.
	  /Y           Подавление запроса подтверждения на перезапись существующего конечного файла.
	  /-Y          Обязательный запрос подтверждения на перезапись существующего конечного файла.
	  /Z           Копирование сетевых файлов с возобновлением.
	  /L           Если источник является символической ссылкой, копирование ссылки вместо реального файла, на который указывает ссылка.

	Чтобы объединить файлы, укажите один конечный и несколько исходных файлов,
	используя подстановочные знаки или формат "файл1+файл2+файл3+...".
		d:\>copy \copy\file.txt \folder\test   - файл \copy\file.txt скопировать в \folder\test
